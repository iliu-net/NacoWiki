Shortocdes for youtube links

This plugin is used to create youtube links from shortcodes.

Link format:

- `[[`
- `youtube:`
- __youtube id__
- Optionally append `^` to open in a new window
- Optionally append `|` and text to show text when hovering
- `]]`
